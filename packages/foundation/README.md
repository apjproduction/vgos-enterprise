# APJ Foundation Runtime

The Foundation Runtime is the TypeScript API for reading, validating, searching, and eventually generating APJ Foundation knowledge.

This package is intentionally lightweight in FP-001C. It provides stable types and pure utility functions so the VGOS app, Advisor, Atlas, and future developer tooling can consume the Foundation Registry without hardcoding platform knowledge.

## Current Scope

- Foundation types
- Registry loading contracts
- Validation helpers
- Search helpers
- Knowledge graph builder

## Future Scope

- Markdown generation
- Atlas integration
- Advisor context provider
- Documentation health scoring
- CLI commands
- CI validation
